Mere Formatter

A live Mere source formatter served by a Mere-authored HTTP backend. Both the page you're reading and the /fmt endpoint below are handled by a single wasm binary compiled from Mere.

(click Format to see the formatted output)

Stack (all Mere-authored, all in this repo):

  • contrib/parser/ — self-host lexer + parser (Phase 50)
  • contrib/fmt/ — self-host formatter (Phase 49)
  • contrib/http/ — Node-hosted HTTP server bindings (Phase 54.35)
  • examples/http_fmt_webapp.mere — this app (imports the three above)
  • examples/http_fmt_webapp.html — the page you're viewing (served by read_file)
